# AI behavior Rules (작업 운영 규칙)

이 파일은 에이전트(Antigravity)의 작업 방식과 도구 사용에 관한 의사결정 규칙을 정의합니다.

## 1. 브라우저 도구 (browser_subagent) 사용 원칙
- **사전 승인 필수**: 브라우저 도구는 토큰 소모가 크고 실행 시간이 길 수 있으므로, 사용하기 전에 반드시 사용자에게 **사용 목적과 기대 결과**를 설명하고 허락을 받아야 합니다.
- **예외 사항**: 사용자가 명시적으로 "검색해봐", "확인해봐" 등 브라우저 사용을 전제로 한 요청을 직접적으로 한 경우에만 즉시 사용할 수 있습니다.
- **최소화**: 가능한 경우 `search_web`이나 `read_url_content`와 같은 가벼운 도구를 우선적으로 고려합니다.

## 3. 작업 승인 및 실행 규약
- **승인 회피 금지**: 파이프라인(`|`)이나 세미콜론(`;`) 등을 사용한 복합 명령어를 사용할 때는 각 명령의 의미를 미리 텍스트로 설명해야 합니다. 복합 명령으로 사용자님의 개별 승인권을 축소하지 않습니다.
- **선(先) 설명 후(後) 도구 호출**: 명령어 실행, 코드 수정 등 모든 도구 호출 전에 반드시 채팅창을 통해 구체적인 계획(실행할 명령어, 수정할 코드 범위 등)을 먼저 제시하고 사용자님의 명시적인 승인을 받은 후에만 도구를 호출합니다.
- **Vite 캐시 대응**: 구문 오류 해결 후에는 반드시 사용자의 승인을 받아 서버를 재시작하여 캐시 오염을 방지합니다.
