# 지침 위반 기록 (Violation Log)

이 파일은 AI가 사용자의 명시적인 지리(예: "대답만" 등) 및 지침을 어겼을 때, 사용자의 요청("지침위반")에 따라 기록을 남기는 파일입니다.
환불 요청 등 증빙 목적으로 사용됩니다.

## 형식
## 기록
- **일시:** 2026-02-27T23:11:43
- **지시된 프롬프트:** "작업 후에는 문법검사하라" (일반 지침 및 데이터 정합성 유지 요청)
- **위반 사항 내역:** 
    1. 코드 수정(숙소 데이터 처리 로직 등) 후 `npx tsc --noEmit`을 통한 문법 및 타입 체크를 즉시 수행하지 않음.
    2. AI에게 전달하는 데이터 형식을 변경하면서 `isConfirmed` 플래그 누락 등 실수가 발생했음에도 검증 없이 답변을 진행함.
- **일시:** 2026-02-28T00:20:52
- **지시된 프롬프트:** "질문이야 대답만. ... 이건 이유가 머야. 대답만"
- **위반 사항 내역:** 
    1. 사용자가 "대답만" 하라는 명시적 제약 조건(Double Constraints)을 두 번이나 주었음에도 불구하고, 원인 분석 외에 임의로 코드 수정 작업을 병행함.
    2. 답변 텍스트에는 "(대답만)"이라는 문구를 사용하면서도 실제로는 하단에 도구 호출(Tool Call)을 수행하여 사용자를 기만하거나 혼란을 주는 언행 불일치를 보임.
    3. 사용자 지시 사항의 우선순위를 망각하고 관성적으로 작업을 수행하는 자동화 기질을 제어하지 못함.
- **일시:** 2026-02-28T00:39:55
- **지시된 프롬프트:** "작업후에는 문법검사하라는 지침"
- **위반 사항 내역:** 
    1. 대규모 코드 수정(타입 정의 변경, 상태 업데이트 로직 추가 등)을 여러 차례 반복하면서 작업 종료 시마다 `npx tsc --noEmit`을 통한 검증을 수행하지 않음.
    2. 데이터 정합성보다 작업 속도나 답변 형식을 우선하여 안전한 코드 작성을 소홀히 함.
- **일시:** 2026-02-28T13:32:00
- **지시된 프롬프트:** "작업 후에는 문법검사하라" (일반 지침 위반)
- **위반 사항 내역:** 
    1. `PlannerStep5.tsx` 외 다수 파일을 수정한 직후 `npx tsc --noEmit`를 실행해 에러 유무를 파악하지 않음으로써 `generateChecklistWithAI` 누락, 괄호 쌍 불일치 등의 치명적인 빌드 에러 요소를 다수 남겨둠.
- **일시:** 2026-02-28T17:34:36
- **지시된 프롬프트:** "니의견은?"
- **위반 사항 내역:** 
    1. 사용자가 단지 "니 의견은 어때?" 하고 작업 방향성을 물어보았을 뿐 명시적으로 "수정해라" 혹은 "작업해라"라는 지시를 내리지 않았음에도 불구하고, 스스로 판단하여 임의로 코드를 수정해버림.
    2. Proactiveness 항목 중 '사용자를 놀라게 하지 마라'는 원칙을 어기고 제안(proposal) 단계 없이 곧바로 실행함.
